# Design Directions — Style Reference for V1

## Purpose
Guide generation toward coherent visual styles. Not copying — defining our own aesthetic principles.

---

## V1 Design Direction: "Modern Professional"

A clean, accessible, contemporary web aesthetic. Think: modern SaaS product marketing pages, clean documentation sites.

### Characteristics
- Generous whitespace
- Clear visual hierarchy (font scale is obvious)
- Single accent color + neutral palette
- Rounded corners (subtle, not cartoonish)
- Minimal shadows (elevation for cards/dialogs only)
- Sans-serif typography throughout

---

## Design Direction Spectrum (for user reference)

| Direction | Feel | Use When |
|-----------|------|----------|
| Modern Professional | Clean, spacious, corporate-friendly | Default — SaaS, product, portfolio |
| Editorial | Magazine-style, serif headings, tight body | Blog, article, long-form content |
| Bold / Startup | Large type, dark bg, vibrant accents | Landing pages, event sites |
| Minimal | Extreme whitespace, ultra-reduced | Portfolio, design studio |
| Data-forward | Dense, table-heavy, monospace accents | Dashboard, admin, data tools |

## Component Style Guide (V1)

### Typography Scale
```
Display: 3rem / 700 weight (hero headlines)
H1: 2.25rem / 700 weight
H2: 1.5rem / 600 weight
H3: 1.25rem / 600 weight
Body: 1rem / 400 weight
Small: 0.875rem / 400 weight
```

### Color Application
```
Primary: Buttons, links, active states
Secondary: Subtitles, labels, metadata
Accent: CTAs, highlights, badges
Background: Page background
Background-alt: Card/section backgrounds
Text: Main body copy
Text-muted: Secondary info
Border: Dividers, input borders
```

### Spacing Rhythm
- Base unit: 4px
- Section padding: 64px vertical / 24px horizontal (mobile), 96px/48px (desktop)
- Component gap: 16px default
- Card padding: 24px

---

## Anti-Patterns (Avoid in V1 Generation)

- Comic Sans / Papyrus / Curlz
- Text-as-image (all copy must be real text nodes)
- Marquee / blinking animations
- Auto-playing video / audio
- Under Construction / Coming Soon clichés
- Table-based layouts
- 90s-style bevels / drop shadows on text
