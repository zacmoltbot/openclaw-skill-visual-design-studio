# openclaw-skill-visual-design-studio

An OpenClaw skill for building, reviewing, and exporting UI design artifacts — from HTML prototypes to polished presentation decks.

## What it does

- **HTML prototypes** — generate live, interactive UI mockups
- **Review packs** — structured feedback bundles with screenshots & annotations
- **Critique reports** — AI-assisted design critique with actionable notes
- **Artifact export** — screenshots, PDFs, and slide decks on demand
- **Artifact diff** — visual comparisons between prototype versions

## Structure

| Path | Purpose |
|------|---------|
| `SKILL.md` | Skill definition and core instructions |
| `scripts/` | Build, export, and diff utilities |
| `references/` | Design tokens, templates, and guidelines |
| `assets/` | Sample assets and output examples |

## Status

Current focus: HTML-first, Playwright-based design workflow.

Built as an original workflow with clean-room methodology — no third-party assets are bundled in this repo.
